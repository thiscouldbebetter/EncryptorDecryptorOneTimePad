One-Time Pad Encryptor/Decryptor

Specify a maximum message size, then click the "Generate" button to generate a psuedorandom key. Send the generated key to a recipient via some secure method, then compose a message and click the "Ecrypt Message with Key" button to encrypt the message. The encrypted message can be sent to the recipient over a unsecure channel. The recipient can then load the key they received earlier along with the encrypted message, then click the "Decrypt Message With Key" button to see the message.
